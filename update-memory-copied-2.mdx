`PUT /memories/{id}`

Actualiza una memoria existente por su ID.

### **Ejemplos de Código**

<CodeGroup>

```http cURL
curl --request PUT \
  --url 'https://api.tess.im/memories/{id}' \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "content": "Contenido actualizado de la memoria",
    "metadata": {
      "source": "fuente_actualizada",
      "tags": ["actualizado", "prueba"]
    }
  }'
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    data := map[string]interface{}{
        "memory":        "Updated memory content",
        "collection_id": 1,
    }
    
    jsonData, _ := json.Marshal(data)
    
    client := &http.Client{}
    req, err := http.NewRequest("PATCH", "https://api.tess.im/memories/{memoryId}", bytes.NewBuffer(jsonData))
    if err != nil {
        fmt.Println(err)
        return
    }
    
    req.Header.Add("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Add("Content-Type", "application/json")
    
    resp, err := client.Do(req)
    if err != nil {
        fmt.Println(err)
        return
    }
    defer resp.Body.Close()
    
    body, err := ioutil.ReadAll(resp.Body)
    if err != nil {
        fmt.Println(err)
        return
    }
    
    fmt.Println(string(body))
}
```

### **Encabezados**

| **Parámetro**  | **Tipo** | **Requerido** | **Descripción**                                                                                               |
| :------------- | :------- | :------------ | :------------------------------------------------------------------------------------------------------------ |
| x-workspace-id | integer  | No            | ID del espacio de trabajo. Si no se proporciona, se utilizará el espacio de trabajo seleccionado del usuario. |

<Note>
  **Nota:** Este campo será requerido en una futura versión de la API. Se recomienda encarecidamente configurarlo ahora para asegurar la compatibilidad con futuras actualizaciones.
</Note>

### **Parámetros de Ruta**

| **Parámetro** | **Tipo** | **Requerido** | **Descripción**     |
| :------------ | :------- | :------------ | :------------------ |
| id            | string   | Sí            | El ID de la memoria |

### **Parámetros del Cuerpo**

| **Parámetro** | **Tipo** | **Requerido** | **Descripción**                        |
| :------------ | :------- | :------------ | :------------------------------------- |
| content       | string   | No            | El contenido actualizado de la memoria |
| metadata      | object   | No            | Metadatos actualizados de la memoria   |
