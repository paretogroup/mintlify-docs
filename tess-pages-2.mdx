Hemos hecho que el proceso de publicar tu sitio web sea más simple y rápido que nunca. Con Tess Pages, puedes crear y publicar tu página en la plataforma (ya sea una landing page, un formulario, dashboard, presentación, etc.), usando el propio chat.

Tess Pages es la funcionalidad de instant cloud de Tess, una solución para publicar páginas con un único comando.

### **¿Qué es el Instant Cloud?**

El Instant Cloud es la herramienta que transforma tus archivos en páginas web activas y accesibles, que llamamos Tess Pages. Con un simple comando, puedes alojar tus documentos directamente en la infraestructura de Tess AI, generando un enlace público para compartir con quien quieras.

Es la forma más rápida y fácil de poner tu contenido en línea, sin necesidad de ningún conocimiento técnico. La complejidad de alojar un sitio web se ha reducido a una simple solicitud.

### **¿Qué puedo publicar?**

Las posibilidades son vastas. Puedes usar el Instant Cloud para publicar:

### **Cómo Funciona: Guía Paso a Paso**

El flujo de trabajo es increíblemente directo. Sigue estos pasos:

<Steps>
  <Step title="Ten el Archivo HTML">
    El primer paso es tener tu archivo .html listo y disponible en el contexto de la conversación. Puedes hacerlo de dos formas:

    - Creando el archivo en el momento: Usa la herramienta Manage Files para escribir el código HTML de un sitio directamente en el chat.
    - Subiendo un archivo existente: Sube tu archivo HTML listo al usar una de las herramientas que permiten adjuntos, como Manage Files o Deep Analysis.

      <Frame>
        ![Tessdocs Pages1](/images/tessdocs-pages1.png)
      </Frame>
  </Step>
  <Step title="Activa la Herramienta">
    En el menú "Tools", selecciona la opción "Manage Files" o "Deep Analysis".

    <Frame>
      ![Tessdocs Pages2](/images/tessdocs-pages2.png)
    </Frame>
  </Step>
  <Step title="Genera el Archivo HTML y Refina el Resultado">
    Pide que se genere el archivo HTML como vista previa en Tess, para que puedas ver el resultado y realizar los ajustes necesarios.
  </Step>
  <Step title="Solicita la Publicación">
    Con el archivo disponible en el chat, simplemente da un comando claro a la IA. No se necesitan configuraciones complejas. Solo pídelo.

    Ejemplos de Comandos:

    - "Publica este archivo HTML como un sitio web."
    - "Crea una Tess Page con el archivo mi-sitio.html."
    - "Quiero publicar este sitio web. Genera el enlace para mí."

      <Frame>
        ![Tessdocs Pages3](/images/tessdocs-pages3.png)
      </Frame>
  </Step>
</Steps>

**¡Listo! ¡Tu Sitio Está en Línea!**

La IA procesará tu solicitud y te proporcionará el enlace público a tu nueva Tess Page. ¡Así de simple! Tu contenido estará instantáneamente accesible para el mundo.

<Note>
  **Información Importante:**

  Para garantizar la mejor experiencia, ten en cuenta dos reglas simples:

  - Subdominio Personalizado: Tienes la libertad de elegir un subdominio personalizado para tu sitio (ej.: [mi-proyecto-increible.tess.page](http://mi-proyecto-increible.tess.page)). Sin embargo, el nombre elegido no puede estar ya en uso en la plataforma. ¡Sé creativo!
  - Un Sitio por Chat: Cada conversación en el chat está limitada a la publicación de un único sitio web. Si publicas un nuevo archivo en el mismo chat, reemplazará al sitio anterior. Para publicar un segundo sitio, simplemente inicia un nuevo chat.
</Note>
