---
title: "Rollback de Versión del Agente"
description: "Restaura una versión específica del agente creando una nueva versión cuyo snapshot coincide con el objetivo. El historial de versiones se preserva — no se elimina ninguna versión. El rollback se registra como una entrada de auditoría. Requiere el permiso `agent:version:read`."
api: "POST https://api.tess.im/agents/{id}/versions/{version_number}/rollback"
authMethod: "bearer"
playground: "interactive"
---


<Note>
  Intentar hacer rollback a la versión actualmente activa devuelve un error `422`.
</Note>

### **Headers**

| **Parámetro**  | **Tipo** | **Requerido** | **Descripción**                                                                             |
| :------------- | :------- | :------------ | :------------------------------------------------------------------------------------------ |
| x-workspace-id | integer  | No            | ID del workspace. Si no se proporciona, se usará el workspace seleccionado por el usuario.  |

<Note>
  **Nota:** Este campo será obligatorio en una versión futura de la API. Se recomienda establecerlo ahora para garantizar compatibilidad con futuras actualizaciones.
</Note>

### **Parámetros de Ruta**

| **Parámetro** | **Tipo** | **Requerido** | **Descripción**                                 |
| :------------ | :------- | :------------ | :---------------------------------------------- |
| id            | integer  | Sí            | El ID del agente                                |
| version_id    | integer  | Sí            | El ID de la versión a la que hacer rollback     |

### **Campos de la Respuesta**

| **Campo**                         | **Tipo**         | **Descripción**                                                                       |
| :-------------------------------- | :--------------- | :------------------------------------------------------------------------------------ |
| message                           | string           | Mensaje de confirmación indicando la versión de origen y la nueva versión activa      |
| version.version_id                | integer          | El nuevo ID de versión creado por este rollback                                       |
| version.published_by              | string           | Email del usuario que realizó el rollback                                             |
| version.published_at              | string (ISO8601) | Timestamp de cuándo se creó la versión de rollback                                   |
| version.rollback_from_version     | integer          | El `version_id` utilizado como origen para este rollback                             |
| version.snapshot                  | object           | Estado completo del agente restaurado por este rollback (misma estructura que Get Agent Version) |

### **Respuestas de Error**

#### **Versión Ya Activa (422)**

```json
{
  "message": "Cannot roll back to the current active version."
}
```

#### **No Encontrado (404)**

```json
{
  "message": "Version not found."
}
```

#### **Prohibido (403)**

```json
{
  "message": "You do not have permission to roll back agent versions."
}
```
