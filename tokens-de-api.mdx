---
title: "Tokens de API"
---

Os Tokens de API são chaves de acesso que permitem que aplicações e serviços externos se comuniquem com a Tess AI de forma segura, possibilitando a criação de integrações e automações personalizadas.

### **Como acessar a área de Tokens de API**

<Step title="Acesse as configurações da conta">
  Faça login na sua conta Tess AI e depois clique no ícone do seu perfil, localizado no canto inferior esquerdo da tela. No menu que se abre, selecione a opção "Configurações".
</Step>

<Step title="Acesse as preferências">
  Localize a opção "Token de API".
</Step>

### **Ações Disponíveis**

Nesta tela, você pode realizar duas ações principais:

1. <u>Gerar um Novo Token:</u> Ao clicar nesta opção, um novo token exclusivo será criado. Você poderá nomeá-lo para facilitar a identificação de onde ele está sendo usado.
2. <u>Gerenciar Tokens Ativos:</u> Você verá uma lista de todos os tokens que já foram gerados. Para cada token, só é possível visualizá-lo assim que é gerado, logo, não é possível ver a chave novamente para copiar seu valor. Nesse caso, conseguimos excluí-la, verificar quem criou e o último uso!

<Warning>
  **_<u>Aviso de Segurança: Trate seus tokens como senhas!</u>_**

  Um Token de API permite o acesso à sua conta Tess AI por meio de outras aplicações. Nunca compartilhe seus tokens publicamente ou os exponha em códigos de front-end (acessíveis pelo navegador). Se um token for comprometido, revogue-o imediatamente e gere um novo.
</Warning>

Se tiver qualquer dúvida sobre o uso da API, nossa equipe de suporte está à disposição para ajudar pelo e-mail: [<u>suporte@tess.im</u>](mailto:suporte@tess.im).
