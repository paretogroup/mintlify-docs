### **Headers**

| **Parameter** | **Type** | **Required** | **Description** |
| :-- | :-- | :-- | :-- |
| x-workspace-id | integer | Yes | Workspace ID |
| Authorization | string | Yes | Bearer Token for authorization |

<Note>
  **Note:** This field will be required in a future API release. It is highly recommended to set it now to ensure compatibility with future updates.
</Note>

### **Parameters**

| **Parameter** | **Type** | **Required** | **Description** |
| :-- | :-- | :-- | :-- |
| range | string | No | Range days |
| page | number | No | Number of the page |
| per\_page | number | No | Number of items per page |
| type | string | No | Types: all, chat, imagem, text, voiceover, video, code |
| start\_date | date | No | Start date, example: 2026-03-31 |
| end\_date | date | No | End date, example: 2026-03-31 |
