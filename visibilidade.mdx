A Visibilidade define o nível de privacidade do agente criado no Agent Studio.

Você pode escolher entre os seguintes níveis:

<Columns cols={2}>
  <Card title="Workspace">
    Disponível apenas para os membros do seu workspace. Esse é o modo ideal para manter o controle de agentes internos.
  </Card>
  <Card title="Privado">
    Visível apenas para você (e, se houver, administradores autorizados do workspace).
  </Card>
  <Card title="Público">
    Disponível para uso por toda a comunidade de usuários Tess  e está listado no Marketplace. 
  </Card>
  <Card title="Não listado">
    Não aparece no Marketplace, mas qualquer pessoa da Tess que receba o link consegue acessar.
  </Card>
</Columns>

### **Por que é importante?**

- Segurança e sigilo: agentes com dados sensíveis devem ser privados ou limitados ao workspace.
- Compartilhamento controlado: agentes experimentais podem ser não listados, acessíveis apenas via link.
- Distribuição e comunidade: agentes maduros podem ser tornados públicos e compartilhados com toda a base Tess.

### Entendendo em detalhes os Níveis de visibilidade

<Card icon="1" title="Workspace">
  Todos os membros do workspace podem acessar o agente. É ótimo para criação de agentes por membros de empresas para que outros colegas possam utilizar. 
</Card>

<Card icon="2" title="Privado">
  Apenas o criador do agente consegue vê-lo. Usado quando se trata de agentes em rascunhos, testes ou que são agentes pessoais.
</Card>

<Card icon="3" title="Público">
  Nesse caso, qualquer usuário da Tess pode localizar e usar o agente, ficando disponível e listado no Marketplace. São usados para agentes prontos para reutilização por terceiros (mais genéricos e amplos), templates de mercado, showcases, etc.
</Card>

<Card icon="4" title="Público">
  Não são possíveis de localizar no Marketplace e apenas as pessoas com o link direto podem abri-lo. É usado quando querem  compartilhar com clientes específicos, parceiros, ou em fases beta para testes por outras pessoas.
</Card>

### **Como ajustar a visibilidade do agente**

No momento de criação do Agente, você pode definir a visibilidade dentro do Agent Studio, selecionando a opção de sua preferência. O agente será criado com aquela orientação.

<Frame>
  ![Image](/images/image-77.png)
</Frame>

Além disso, você sempre pode ajustar o nível de visibilidade no Agent Studio:

1. Abra o Agent Studio
2. Localize o agente que deseja configurar. Pode ser pelo campo de busca, o tipo, visibilidade ou dono.
3. Veja a coluna Visibilidade e escolha o nível desejado

<Warning>
  **IMPORTANTE**\
  \
  Se optar por tornar público seu agente, revise nome, descrição e conteúdo antes de expor ao Marketplace. O nosso time analisará e testará seu agente antes de torná-lo definitivamente público. Se for reprovado, eles sinalizarão os motivos para ajuste e nova verificação.
</Warning>

<Tip>
  **Boas práticas**

  - Comece com o agente como Privado durante a fase de testes.
  - Quando estiver maduro, mude para Workspace para todo o time usar.
  - Só torne Público o que você realmente quer compartilhar com a comunidade, sem dados sensíveis.
  - Use Não listado para pilotos com clientes selecionados, sem expor no Marketplace.
</Tip>
