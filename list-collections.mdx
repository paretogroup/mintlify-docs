`GET /memory-collections`

Retrieves a paginated list of memory collections for the authenticated user.

### **Code Examples**

<CodeGroup>

```http cURL
curl --request GET \
  --url 'https://api.tess.im/memory-collections?page=1&per_page=10&search=example' \
  --header 'Authorization: Bearer YOUR_API_KEY'
```

```go Go
package main

import (
    "fmt"
    "io/ioutil"
    "net/http"
)

func main() {
    client := &http.Client{}
    req, err := http.NewRequest("GET", "https://api.tess.im/memory-collections?page=1&per_page=10&search=example", nil)
    if err != nil {
        fmt.Println(err)
        return
    }

    resp, err := client.Do(req)
    if err != nil {
        fmt.Println(err)
        return
    }
    defer resp.Body.Close()
    
    body, err := ioutil.ReadAll(resp.Body)
    if err != nil {
        fmt.Println(err)
        return
    }
    
    fmt.Println(string(body))
}
```

### **Query Parameters**

| **Parameter** | **Type** | **Required** | **Description**                   | **Default** | **Min** | **Max** |
| :------------ | :------- | :----------- | :-------------------------------- | :---------- | :------ | :------ |
| page          | integer  | No           | Page number                       | 1           | -       | -       |
| per_page      | integer  | No           | Items per page                    | 10          | 1       | 50      |
| search        | string   | No           | Search term to filter collections | -           | -       | -       |
