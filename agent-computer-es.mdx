Tess Agent Computer es un General AI Agent autónomo que va más allá de las simples conversaciones con IA. En lugar de gestionar prompts de forma individual, Tess Agent Computer asume trabajos complejos y de múltiples pasos, llevándolos de principio a fin.

Dile a Tess lo que necesitas, aléjate y vuelve al trabajo completado — documentos pulidos, archivos organizados, investigaciones consolidadas, hojas de cálculo con fórmulas completamente funcionales y mucho más.

## ¿Qué es Tess Agent Computer?

Tess Agent Computer cambia la forma en que trabajas con la IA. Mientras que las conversaciones com chatbots de IA requieren tu presencia en cada etapa, Agent Computer trabaja por su cuenta — planificando, ejecutando y entregando resultados sin que tengas que supervisar cada acción.

Este nivel de ejecución autónoma no está disponible en conversaciones regulares fuera de Tess.

## Capacidades principales

<Accordion title="Orquestación multi-agente y multi-modelo">
  Tess Agent divide los trabajos complejos en subtareas más pequeñas y gestiona flujos de trabajo paralelos para completarlos de manera eficiente. Cada paso se asigna al modelo mejor preparado para ese tipo de trabajo.
</Accordion>

<Accordion title="Entregables profesionales">
  Produce resultados listos para usar — hojas de cálculo de Excel con fórmulas funcionales, presentaciones de PowerPoint, documentos formateados y más — sin ningún esfuerzo manual adicional.
</Accordion>

<Accordion title="Tareas de larga duración">
  Afronta trabajos complejos durante períodos prolongados sin ser interrumpido por tiempos de espera de sesión o limitaciones de contexto.
</Accordion>

<Accordion title="Tareas programadas">
  Configura y guarda tareas para ejecutarlas bajo demanda o con una programación recurrente de tu elección — diaria, semanal o personalizada.
</Accordion>

<Accordion title="Autopilot">
  Tess Agent sigue funcionando incluso si cierras la pestaña, pierdes la conexión o apagas tu dispositivo. Autopilot garantiza hasta 50 minutos de ejecución ininterrumpida, sin necesidad de supervisión.
</Accordion>

## Cómo Tess Agent ejecuta tus tareas

El código se ejecuta de forma segura dentro de un entorno aislado, pero Tess es capaz de realizar cambios reales en tus archivos y sistemas.

Una vez que inicias una tarea en Agent Computer, Tess:

1. Revisa tu solicitud y elabora un plan de ejecución.
2. Divide el trabajo en subtareas cuando sea necesario, manteniendo la eficiencia.
3. Ejecuta todo dentro de una máquina virtual (VM) segura y aislada.
4. Gestiona múltiples flujos de trabajo de forma simultánea, cuando tiene sentido hacerlo.
5. Entrega los resultados finales directamente a tu sistema de archivos.

Te mantienes completamente informado de lo que Tess está planificando y haciendo en cada etapa. Puedes intervenir para dar orientación cuando sea necesario — o simplemente dejar que Tess se encargue de todo de forma independiente.

<Info>
  En ciertos momentos, Tess Agent puede hacer algunas preguntas para asegurarse de que el resultado final sea exactamente lo que tenías en mente.
</Info>
