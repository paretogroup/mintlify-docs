Comece a usar a API Tess AI em poucos minutos.

## **Criar uma Chave de API**

Para usar a API Tess AI, você precisará de uma chave de API. Você pode criar uma:

1. Indo diretamente para [#Tess AI → Tokens de Usuário](https://tess.im/dashboard/user/tokens)
2. Ou navegando pela interface:
   - Visite [#Tess AI](https://tess.im/)
   - Clique no Menu do Usuário
   - Selecione "Tokens de API"
   - Clique em "Adicionar Novo Token"

## **Configurar sua Chave de API (recomendado)**

Configure sua chave de API como uma variável de ambiente. Esta abordagem simplifica o uso da API eliminando a necessidade de incluir sua chave de API em cada requisição. Além disso, aumenta a segurança minimizando o risco de incluir inadvertidamente sua chave de API em seu código.
