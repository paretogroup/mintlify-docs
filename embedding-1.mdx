Pon tu agente de Tess disponible directamente dentro de sitios web y aplicaciones, como un widget de chat o componente embebido. Ideal para atención al cliente, captura de leads, onboarding y asistentes internos.

### **¿Qué es?**

El recurso de Embedding permite incorporar (embeber) un agente de Tess en:

- sitios web institucionales
- áreas de acceso restringido de clientes
- productos / SaaS propios
- intranets y portales internos

Tess genera un código de incorporación (generalmente un script o iframe) para que lo pegues en tu sitio web/app. A partir de ahí, tus usuarios pueden conversar con el agente directamente en esa interfaz.

### Por qué es importante

<AccordionGroup>
  <Accordion title="Experiencia nativa">
    El usuario no necesita salir de tu sitio web para hablar con el agente.
  </Accordion>
  <Accordion title="Soporte 24/7">
    Agentes de soporte, FAQ y onboarding siempre disponibles.
  </Accordion>
  <Accordion title="Conversión">
    Los agentes comerciales ayudan a calificar y orientar leads en tiempo real.
  </Accordion>
  <Accordion title="Flexibilidad">
    Un mismo agente puede usarse en múltiples canales (sitio web, app, interno y externo).
  </Accordion>
</AccordionGroup>

### ¿Cómo generar un código de embedding?

<Steps>
  <Step title="Crea el agente">
    El primer paso es la creación y prueba del agente (prompt, Knowledge Base, Tools, etc.).
  </Step>
  <Step title="Ajusta la visibilidad">
    Configura la visibilidad, ya que tu agente no puede ser privado para habilitar el Embed.
  </Step>
  <Step title="Accede a la Configuración detallada del agente">
    En la lista de agentes del Agent Studio, pasa el cursor junto al estado y haz clic en el lápiz para acceder a la configuración detallada del agente.

    <Frame>
      ![Image](/images/image-79.png)
    </Frame>
    Desplázate hasta el final y activa la opción "Conecta Tu Agente".

    <Frame>
      ![Image](/images/image-80.png)
    </Frame>
  </Step>
  <Step title="Elige el formato">
    Elige el formato de incorporación y utiliza el código en el HTML de tu sitio web/app.

    <Frame>
      ![Image](/images/image-81.png)
    </Frame>
    De esta forma, el widget o componente de chat comenzará a aparecer para los visitantes, conectado a tu agente.
  </Step>
</Steps>

<Tip>
  **Buenas prácticas**

  - Comienza incorporando en entornos de prueba/staging antes de ir a producción.
  - Asegúrate de que el agente esté configurado y probado para uso externo.
  - Verifica que las políticas de privacidad de tu empresa cubran las interacciones con agentes de IA.
</Tip>
