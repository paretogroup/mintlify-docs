---
title: "Modo Max"
---

Nossa funcionalidade para ativar as maiores janelas de contexto disponíveis, o que resulta em desempenho mais lento e custos mais altos

Normalmente, a Tess usa uma janela de contexto de 32k tokens (~24.000 palavras). O Modo Max permite que você habilite as maiores janelas de contexto disponíveis para todos os modelos suportados. Isso é especialmente útil para conversas longas e para modelos que suportam contexto estendido, como 200k tokens nos modelos Claude ou 1M tokens no GPT-4.1 e Gemini 2.5 Pro.

> **Info:** O preço do Modo MAX é calculado com base em tokens, baseado no preço da API do provedor do modelo.

Observe que usar o Modo Max pode ser mais lento e mais caro.
