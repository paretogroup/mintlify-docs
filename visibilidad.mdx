La Visibilidad define el nivel de privacidad del agente creado en Agent Studio. Puedes elegir entre los siguientes niveles:

<Columns cols={2}>
  <Card title="Workspace">
    Disponible solo para los miembros de tu workspace. Este es el modo ideal para mantener el control de agentes internos.
  </Card>
  <Card title="Privado">
    Visible solo para ti (y, si aplica, administradores autorizados del workspace).
  </Card>
  <Card title="Público">
    Disponible para uso por toda la comunidad de usuarios Tess y está listado en el Marketplace.
  </Card>
  <Card title="No listado">
    No aparece en el Marketplace, pero cualquier persona de Tess que reciba el link puede acceder.
  </Card>
</Columns>

<Frame>
  ![Image](/images/image-77.png)
</Frame>

### **¿Por qué es importante?**

- Seguridad y confidencialidad: agentes con datos sensibles deben ser privados o limitados al workspace.
- Compartir de forma controlada: agentes experimentales pueden ser no listados, accesibles solo por link.
- Distribución y comunidad: agentes maduros pueden hacerse públicos y compartirse con toda la base de usuarios de Tess.

### Entendiendo en detalle los niveles de Visibilidad

<Card icon="1" title="Workspace">
  Todos los miembros del workspace pueden acceder al agente. Es excelente para crear agentes por miembros de empresas para que otros colegas los puedan usar.
</Card>

<Card icon="2" title="Privado">
  Solo el creador del agente puede verlo. Se usa cuando se trata de agentes en borradores, pruebas o agentes personales.
</Card>

<Card icon="3" title="Público">
  En este caso, cualquier usuario de Tess puede localizar y usar el agente, quedando disponible y listado en el Marketplace. Se usan para agentes listos para reutilización por terceros (más genéricos y amplios), templates de mercado, showcases, etc.
</Card>

<Card icon="4" title="No listado">
  No se puede localizar en el Marketplace y solo las personas con el link directo pueden abrirlo. Se usa cuando quieres compartir con clientes específicos, partners, o en fases beta para pruebas por otras personas.
</Card>

### **Cómo ajustar la visibilidad del agente**

Al momento de crear el Agente, puedes definir la visibilidad dentro de Agent Studio, seleccionando la opción de tu preferencia. El agente se creará con esa orientación.

<Frame>
  ![Image](/images/image-77.png)
</Frame>

Además, siempre puedes ajustar el nivel de visibilidad en Agent Studio:

1. Abre Agent Studio
2. Ubica el agente que quieres configurar. Puede ser por el campo de búsqueda, el tipo, la visibilidad o el Owner.
3. Revisa la columna Visibilidad y elige el nivel deseado

<Warning>
  **IMPORTANTE**\
  \
  Si eliges hacer público tu agente, revisa el nombre, la descripción y el contenido antes de exponerlo en el Marketplace. Nuestro equipo analizará y probará tu agente antes de hacerlo definitivamente público. Si es rechazado, te señalarán los motivos para ajustar y volver a verificar.
</Warning>

<Tip>
  **Buenas prácticas**

  - Empieza con el agente como Privado durante la fase de pruebas.
  - Cuando esté maduro, cámbialo a Workspace para que lo use todo el equipo.
  - Solo haz Público lo que de verdad quieres compartir con la comunidad, sin datos sensibles.
  - Usa No listado para pilotos con clientes seleccionados, sin exponerlo en el Marketplace.
</Tip>
