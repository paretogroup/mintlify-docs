Tornamos o processo de publicar seu site na web mais simples e rápido do que nunca. Com a nova funcionalidade de publicação direta, você pode criar sua Tess Page (seja uma landing page, um formulário ou uma apresentação) diretamente do seu chat, sem a necessidade de módulos ou recursos avançados para isso.

A Tess AI tem o prazer de anunciar o Tess Page, ou Instant Cloud, a sua solução de publicação com um único comando.

### **O que é o Instant Cloud?**

O Instant Cloud é a ferramenta que transforma seus arquivos em páginas da web ativas e acessíveis, que chamamos de Tess Pages. Com um simples comando, você pode hospedar seus documentos diretamente na infraestrutura da Tess AI, gerando um link público para compartilhar com quem você quiser.

É a maneira mais rápida e fácil de colocar seu conteúdo online, sem nenhum conhecimento técnico necessário. A complexidade de hospedar um site foi reduzida a um simples pedido.

### **O que eu posso publicar?**

As possibilidades são vastas. Você pode usar o Instant Cloud para publicar:

### **Como Funciona: Um Guia Passo a Passo**

O fluxo de trabalho é incrivelmente direto. Siga estes três passos:

<Steps>
  <Step title="Tenha o Arquivo HTML">
    O primeiro passo é ter seu arquivo .html pronto e disponível no contexto da conversa. Você pode fazer isso de duas formas:

    - Criando o arquivo na hora: Use a ferramenta Manage Files para escrever o código HTML de um site diretamente no chat.
    - Enviando um arquivo existente: Faça o upload do seu arquivo HTML pronto ao usar uma das ferramentas que permitem anexos, como Manage Files ou Deep Analysis.

      <Frame>
        ![Tessdocs Pages1](/images/tessdocs-pages1.png)
      </Frame>
  </Step>
  <Step title="Ative a Ferramenta">
    No menu "Tools", selecione a opção "Manage Files" ou "Deep Analysis”.

    <Frame>
      ![Tessdocs Pages2](/images/tessdocs-pages2.png)
    </Frame>
  </Step>
  <Step title="Gere o arquivo HTML e Refine o resultado">
    Peça para que seja gerado o arquivo HTML em visualização na Tess, assim você poderá ver o resultado como prévia e fazer quaisquer ajustes necessários.
  </Step>
  <Step title="Solicite a Publicação">
    Com o arquivo disponível no chat, basta dar um comando claro para a IA. Não precisa de configurações complexas. Apenas peça.

    Exemplos de Comandos:

    - “Publique este arquivo HTML como um site.”
    - “Crie uma Tess Page com o arquivo meu-site.html.”
    - “Quero publicar este site. Gere o link para mim.”

      <Frame>
        ![Tessdocs Pages3](/images/tessdocs-pages3.png)
      </Frame>
  </Step>
</Steps>

**Pronto! Seu Site Está no Ar!**

A IA irá processar seu pedido e fornecerá o link público para a sua nova Tess Page. Simples assim! Seu conteúdo estará instantaneamente acessível para o mundo.

<Note>
  **Informações Importantes:**

  Para garantir a melhor experiência, fique atento a duas regras simples:

  - Subdomínio Personalizado: Você tem a liberdade de escolher um subdomínio personalizado para o seu site (ex: [meu-projeto-incrivel.tess.page](http://meu-projeto-incrivel.tess.page)). No entanto, o nome escolhido não pode já estar em uso na plataforma. Seja criativo!
  - Um Site por Chat: Cada conversa no chat está limitada à publicação de um único site. Se você publicar um novo arquivo no mesmo chat, ele substituirá o site anterior. Para publicar um segundo site, basta iniciar um novo chat.
</Note>
