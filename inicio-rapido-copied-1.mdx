Comienza a usar la API de Tess AI en pocos minutos.

### **Crear una Clave de API**

Para usar la API de Tess AI, necesitarás una clave de API. Puedes crear una:

1. Yendo directamente a [#Tess AI → Tokens de Usuario](https://tess.im/dashboard/user/tokens)
2. O navegando por la interfaz:
   - Visita [#Tess AI](https://tess.im/)
   - Haz clic en el Menú de Usuario
   - Selecciona "Tokens de API"
   - Haz clic en "Agregar Nuevo Token"

### **Configurar tu Clave de API (recomendado)**

Configura tu clave de API como una variable de entorno. Este enfoque simplifica el uso de la API eliminando la necesidad de incluir tu clave de API en cada solicitud. Además, aumenta la seguridad minimizando el riesgo de incluir inadvertidamente tu clave de API en tu código.
