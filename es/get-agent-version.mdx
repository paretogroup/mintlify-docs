`GET /agents/{id}/versions/{version_id}`

Devuelve el snapshot completo de una versión específica del agente, incluyendo todos los campos de configuración en el momento en que esa versión fue guardada. Requiere el permiso `agent:version:read`.

### **Headers**

| **Parámetro**  | **Tipo** | **Requerido** | **Descripción**                                                                             |
| :------------- | :------- | :------------ | :------------------------------------------------------------------------------------------ |
| x-workspace-id | integer  | No            | ID del workspace. Si no se proporciona, se usará el workspace seleccionado por el usuario.  |

<Note>
  **Nota:** Este campo será obligatorio en una versión futura de la API. Se recomienda establecerlo ahora para garantizar compatibilidad con futuras actualizaciones.
</Note>

### **Parámetros de Ruta**

| **Parámetro** | **Tipo** | **Requerido** | **Descripción**                     |
| :------------ | :------- | :------------ | :---------------------------------- |
| id            | integer  | Sí            | El ID del agente                    |
| version_id    | integer  | Sí            | El ID de la versión a consultar     |

### **Campos de la Respuesta**

| **Campo**                         | **Tipo**         | **Descripción**                                                                         |
| :-------------------------------- | :--------------- | :-------------------------------------------------------------------------------------- |
| version.version_id                | integer          | Identificador secuencial único de esta versión                                          |
| version.published_by              | string           | Email del usuario que guardó esta versión                                               |
| version.published_at              | string (ISO8601) | Timestamp de cuándo se creó esta versión                                                |
| version.rollback_from_version     | integer \| null  | Si fue creada por un rollback, el `version_id` de origen; de lo contrario `null`       |
| version.snapshot.agent_name       | string           | Nombre del agente en el momento en que se guardó esta versión                           |
| version.snapshot.instructions     | string           | Prompt / instrucciones del sistema en el momento en que se guardó esta versión          |
| version.snapshot.ask_user_questions | array          | Campos de entrada del usuario configurados en el momento en que se guardó esta versión  |
| version.snapshot.steps            | array            | Steps de automatización configurados en el momento en que se guardó esta versión        |
| version.snapshot.knowledge_base_files | array        | Archivos de base de conocimiento vinculados en el momento en que se guardó esta versión |
| version.snapshot.type             | string           | Tipo del agente (ej: `chat`)                                                            |
| version.snapshot.subtype          | string           | Subtipo del agente (ej: `All LLM Models`)                                               |
| version.snapshot.advanced_settings | object          | Configuración de modelo y temperatura en el momento en que se guardó esta versión       |
| version.snapshot.visibility       | string           | Visibilidad del agente (`private` o `public`) en el momento en que se guardó esta versión |

### **Respuestas de Error**

#### **No Encontrado (404)**

```json
{
  "message": "Version not found."
}
```

#### **Prohibido (403)**

```json
{
  "message": "You do not have permission to view agent versions."
}
```
