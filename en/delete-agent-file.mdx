### **Headers**

| **Parameter**  | **Type** | **Required** | **Description**                                                                                       |
| :------------- | :------- | :-------------- | :-------------------------------------------------------------------------------------------------- |
| x-workspace-id | integer  | No             | Workspace ID. If not provided, the user's selected Workspace will be used. |

<Note>
  **Note:** This field will be required in a future API release. It is highly recommended to set it now to ensure compatibility with future updates.
</Note>

### **Parâmetros de Rota**

| **Parameter** | **Type** | **Required** | **Description**                |
| :------------ | :------- | :-------------- | :--------------------------- |
| agentId       | string   | Yes             | The agent ID              |
| fileId        | string   | Yes             | The ID of the file to delete |
